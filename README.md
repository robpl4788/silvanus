# Silvanus

A logging and analysis platform designed for lightweight robots and embedded systems.

Flutter front end
Rust back end

## Aims to:

### Read:
 - CSV
 - Serial Printouts
 - Custom log files

### Plot
 - Line
 - Statistics
 - 2d and 3d poses in custom environments

 ## Name:
 Silvanus is roman god of forests, logging happens in forests.
